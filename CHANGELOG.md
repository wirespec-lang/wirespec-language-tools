# Changelog

## 0.1.0

Initial release.

- TextMate grammar for `.wspec` files — keywords, types, annotations, operators, comments, literals
- LSP client for `wirespec-lsp` — diagnostics, go to definition, hover, completion, document symbols, semantic tokens
- 12 code snippets for common patterns (packet, frame, capsule, state machine, enum, etc.)
- Bracket matching, auto-closing, code folding, comment toggling
